# Note: For full GitHub integration, install these system tools separately:
# - Git (https://git-scm.com/downloads)
#   - Debian:
#     1. Open a terminal.
#     2. Update package list: `sudo apt update`
#     3. Install Git: `sudo apt install git`
#     4. Verify installation: `git --version`
#   - Fedora:
#     1. Open a terminal.
#     2. Update package list: `sudo dnf update`
#     3. Install Git: `sudo dnf install git`
#     4. Verify installation: `git --version`

# - GitHub CLI (gh) (https://cli.github.com/)
#   - Debian:
#     1. Open a terminal.
#     2. Add GitHubâ€™s APT key: `curl -fsSL https://cli.github.com/packages/githubcli-archive-keyring.gpg | sudo dd of=/usr/share/keyrings/githubcli-archive-keyring.gpg`
#     3. Set key permissions: `sudo chmod go+r /usr/share/keyrings/githubcli-archive-keyring.gpg`
#     4. Add GitHub CLI repository: `echo "deb [arch=$(dpkg --print-architecture) signed-by=/usr/share/keyrings/githubcli-archive-keyring.gpg] https://cli.github.com/packages stable main" | sudo tee /etc/apt/sources.list.d/github-cli.list > /dev/null`
#     5. Update package list: `sudo apt update`
#     6. Install GitHub CLI: `sudo apt install gh`
#     7. Verify installation: `gh --version`
#   - Fedora:
#     1. Open a terminal.
